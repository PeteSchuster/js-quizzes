Data Attributes

Data attributes are a good way to store information that can be used later.

Goals:
  1. Update each character block to include an image using the data-image string URL.
  2. The newly created image should use the data-name as its alt text.
  3. The image should appear at the top of the source order in the HTML of the blocks, it should be floated left, include margin and a border.
